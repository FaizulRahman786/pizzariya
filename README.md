# Pizzariya Town — Premium Reusable Frontend Template

A premium, highly interactive, and responsive white-label frontend template designed specifically for sweet shops, bakeries, cafes, and local restaurants. Created with a focus on rich aesthetics, premium user animations, Core Web Vitals performance, and marketplace quality, this template is ready to sell for ₹15,000–₹25,000+.

---

## 🌟 Key Features

- **White-Label Customization Engine**: Customize the entire template (contacts, colors, hours, social links, SEO details, trust badges) by editing a single central configuration file (`config.js`).
- **Dynamic Layout Loader (`layout.js`)**: Modularity at the frontend level. Shared elements like the navigation bar, footer, preloader, WhatsApp widget, and mobile action bar are separated into reusable components (`/components`) and loaded asynchronously.
- **Hindi & English Translation Switcher**: Built-in, lightweight localization system (`translations.js`) that translates UI buttons, headings, and labels dynamically without breaking layout structures.
- **Premium Local Business Features**:
  - **Mobile Sticky Action Bar**: Bottom action bar optimized for mobile users to quickly Call, WhatsApp, or navigate via Google Maps.
  - **Dynamic WhatsApp Product Share**: Users can share specific dishes/sweets on WhatsApp with customized product details and pricing.
  - **Premium Printed Menu**: Elegant print styling (`@media print`) that formats the menu page into a clean, professional print document.
  - **Trust Badges Grid**: Display customizable trust badges (e.g., FSSAI, Pure Veg, Legacy) managed directly from the central configuration.
- **Core Web Vitals Optimized**:
  - WebP image formatting and layout shifts prevention (CLS stabilization).
  - High-performance GPU animations throttled on mobile devices to preserve frames and interactivity.
- **SEO & Schema Integration**:
  - Autogenerated JSON-LD `LocalBusiness` / `Restaurant` schema tags injected directly from `config.js`.
  - Production-ready `robots.txt` and `sitemap.xml` templates included.
- **WCAG Accessibility Hardening**: Enhanced keyboard accessibility, aria-labels, focus indicators, and semantic HTML structure.

---

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6)
- **Styling**: Vanilla CSS3 (Custom Variables layout framework)
- **Icons**: Font Awesome 6.4.0 (CDN Loaded)

---

## 🚀 Getting Started

1. Clone or download this template files into your project directory.
2. Open `config.js` to customize the business details, brand colors, and SEO meta tags.
3. Serve using any local web server (e.g. VS Code Live Server, Python's simple HTTP server, or Nginx).
